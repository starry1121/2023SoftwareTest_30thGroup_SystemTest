# 兼饼果职——前端代码及系统测试代码

## 前端

> 进入main分支

### 1.安装依赖

```
npm install
```

### 2.运行项目

```
npm run serve
```

## 系统测试

> SystemTestCypress分支为修改缺陷后的版本
>
> bug分支为缺陷存在的版本

### 1.安装依赖

```
npm install cypress --save-dev
```

### 2.运行系统测试脚本

```
npm run test:e2e(图形化界面)
npm run test.headless:e2e(控制台输出)
```
